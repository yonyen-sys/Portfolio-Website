# Personal Portfolio Website

A Vue 2 personal portfolio website built with responsive design, PWA support, and a client-side PDF export feature.

## Features

- Hero/About section with profile details
- Skills section with polished visual indicators
- Projects section with technology stack and links
- Contact section with email and social links
- Download PDF button for a printable résumé-style export
- Progressive Web App support with manifest and service worker
- GitHub Pages deployment configuration

## Local development

```bash
npm install
npm run serve
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

## PWA Notes

- Installable on mobile and desktop via browser install prompts
- Works offline for cached assets
- Manifest and service worker support are included
